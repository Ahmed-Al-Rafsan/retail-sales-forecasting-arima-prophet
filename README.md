# \# Retail Sales Forecasting using ARIMA and Prophet



This project explores how time-series forecasting can be used to analyse historical retail sales data and predict future demand patterns.



The objective of the project is to compare two widely used forecasting models — \*\*ARIMA\*\* and \*\*Facebook Prophet\*\* — and evaluate their performance in predicting retail sales.



This repository contains a high-level overview of the project, exploratory data analysis insights, and visualisations used during the analysis.



\---



\## Project Overview



Retail businesses generate large volumes of transactional data every day. Analysing historical sales patterns can help organisations anticipate future demand and make better operational decisions.



In this project, retail sales data was analysed using time-series techniques to identify patterns and generate future sales forecasts.



The analysis follows a structured data analytics workflow:



Data Collection  

→ Data Cleaning  

→ Exploratory Data Analysis  

→ Forecasting Model Development  

→ Model Evaluation  

→ Business Insights



\---



\## Exploratory Data Analysis (EDA)



Exploratory Data Analysis was conducted to understand the structure and behaviour of the dataset.



The analysis focused on:



\- historical sales trends over time

\- regional sales performance

\- identification of fluctuations and demand patterns



These insights helped prepare the dataset for forecasting models.



\---



\## Forecasting Models



Two forecasting models were explored in this project.



\### ARIMA (AutoRegressive Integrated Moving Average)



ARIMA is a classical statistical forecasting model that predicts future values based on patterns in past observations. It is commonly used in economic and demand forecasting.



\### Prophet



Prophet is a forecasting model developed by Meta designed for business time-series data. It automatically detects trends and seasonal patterns such as weekly and yearly cycles.



\---



\## Model Evaluation



The models were evaluated using common forecasting accuracy metrics:



\- Mean Absolute Error (MAE)

\- Root Mean Squared Error (RMSE)

\- R² Score



These metrics help measure prediction error and compare model performance.
